# WebSocket Performance Benchmark

A high-concurrency WebSocket message broker and benchmarking tool built with Python's `asyncio`. This project measures system throughput and latency across multiple concurrent connections.

 🚀 Features

* **Asynchronous Broker:** Handles thousands of connections using non-blocking I/O.
* **Thread-Safe Metrics:** Uses `asyncio.Lock` for accurate response tracking.
* **CLI Support:** Fully configurable via command line arguments.
* **Interactive Client:** Real-time manual testing script.

🛠️ Tech Stack

* **Language:** Python 3.10+
* **Library:** `websockets`, `asyncio`
* **Pattern:** Asynchronous Event Loop
